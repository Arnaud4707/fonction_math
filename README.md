Fonction Math
## Pour clone le depot avec les sub modules
git clone --recurse-submodules git@github.com:Arnaud4707/fonction_math.git

## Utilisation

Pour compiler
```
make
```
ou
```
make all
```

Pour clean sans supprimer l'executable
```
make clean
```

Pour clean tout
```
make fclean
```

Pour clean et complier
```
make re
```
